# k

基于electron的redis客户端，可以在linux、macOs、win下运行


#### features
- [x] redis订阅发布
- [x] 提供命令行工具
- [ ] 黑色主题

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# run unit & end-to-end tests
npm test


```
